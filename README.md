# Sign Edges
## Install

1. Create venv
`python3.10 -m venv .venv`

2. Activate venv
`source .venv/bin/activate`

3. Install dependencies
`pip install -r requirements.txt`

## Usage

- Get help
`sign-edge --help`

- Transform signing images located in \<src> and save in, newly created, \<dst>
`sign-edge --src <src> --dst <dst>`
